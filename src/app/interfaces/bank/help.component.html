Key Action
F1 Help window
F2 Editor window
TAB Switch between inventory, equipment and BoB tabs
drag + CTRL Clone item slot to another tab or slot
drag Move item to another slot, tab, inventory or BoB
DELETE Remove item placeholder
BACKSPACE Deposit whole inventory, equipment or BoB
any letter Initiate search
ESC Clear search
- Withdraw one item on which mouse is hovering
+ Deposit one item on which mouse is on
click Withdraw/Deposit/Equip one
click + ALT Inverted Withdraw/Deposit/Equip one
click + SHIFT Withdraw/Deposit/Equip ALL
click + ALT + SHIFT Inverted Withdraw/Deposit/Equip ALL
any number Withdraw/Deposit X item on which mouse is on, where X is the number you type
click + CTRL If clicked on equipment slot it will search for items that can be put in that slot
